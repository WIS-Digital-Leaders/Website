➕ How to add a new event

Adding a new event is one step only — no editing of events.json required.

  1. Create a new .txt file in events/data/ — use the event title as the filename (lowercase, hyphens instead of spaces, e.g. my-new-event.txt).
  2. Put the event metadata as frontmatter at the top of the file, followed by ---, then the description:
date: YYYY-MM-DD
title: My New Event
category: Workshop
cancelled: false
---
A short description of the event.

Push the file — a GitHub Actions workflow automatically rescans all .txt files and rebuilds events.json. ✅
